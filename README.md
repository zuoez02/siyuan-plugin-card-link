# Siyuan Plugin Card Link

思源卡片链接插件, developed by Z.

## 使用方法

使用斜杠菜单，在底部找到“插入卡片链接”，按回车后。输入网址，点击保存，稍等即可爬取完成。

## 原理

基于思源的HTML块，使用了microlink提供的功能，每日限制50次
